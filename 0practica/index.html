Construcción

La construcción de los circuitos sí se elaboró por cada alumno, así como también se tomó video del funcionamiento de cada circuito para comprobar que las conexiones y el código estvieran correctos. Además, se pudo comprobar las diferencias entre cada código (cuando hubieran simlilitudes) para determinar cómo es el código para hacer diferentes acciones y poder implementarlo en un futuro. Sin embargo, no se tomó imagen de cada circuito como tal, sino que únicamente se tomó el video del circuito en funcionamiento. Es por ello que se muestran en la sección de resultados.

- Resultados -
