- Resumen -

La práctica 1 de este curso tuvo como objetivo repasar los conceptos de electrónica vistos hasta ahora en la carrera. Entre estos conceptos se encuentran las conexiones de LEDs, push buttons, display de 7 segmentos y resistencias para elaborar un circuito que cumpla con el objetivo deseado. De igual manera, se buscó repasar el código que hace funcionar dichos circuitos, por lo que se utilizaron palabras de cógido para dar valores a variables y dejar pasar tiempos. Concluyendo, se cumplieron los objetivos, dado que se logró que todos los circuitos de los ejercicios funcionaran. Además, todos los componentes se identificaron correctamente a la hora de conectarlos, así como se buscaron soluciones en caso de que algo no pudiera quedar como en las instrucciones.

- Introducción -

Los componentes electrónicos son dispositivos que se emplean en los circuitos para conducir o manejar la energía. Se pueden clasificar en dos: pasivos y activos. Los componentes pasivos son aquellos que no dan energía al circuito, pero consumen energía; mientras que los activos son los que modifican o amplifican las señales dentro de un circuito. Ambos tipos de componentes trabajan en conjunto para tener un circuito funcional donde las señales se transmitan y no se queme ningún componente [1].

Uno de los componentes de esta práctica es la resistencia, la cual tiene como objetivo atenuar la corriente que circula por el circuito. También están los diodos, en este caso los LEDs, que permiten el paso de la corriente únicamente en una dirección emitiendo luz al recibir la energía. Asimismo, los push buttons son un tipo de interruptor que cierran el circuito momentáneamente para mandar una señal por el tiempo que este permanezca oprimido. Finalmente, el display de 7 segmentos sirve para visualizar información, siendo que debe de establecerse un código para poder expresar la señal de manera correcta [1].

Por otra parte, Arduino es una plataforma de creación electrónica de código abierto con hardware y software que trabajan en conjunto para tener un circuito que cumpla con las funciones deseadas. Es por ello que el objetivo de Arduino es que cualquier persona o empresa pueda crear sus propios circuitos y pasarlos a placas. De la misma manera, existen softwares que permiten hacer simulaciones tomando el hardware y software de Arduino, para verificar el funcionamiento de los circuitos incluso antes de conectarlos en físico [2].

- Conclusiones -

En conclusión, el objetivo de la práctica se cumplió, dado que se repasaron los conceptos de electrónica. Esto se reflejó en la construcción de los circuitos, ya que todos funcionaron correctamente. Sin embargo, hubo problemas para hacer funcionar los circuitos del display. Esto es porque los tipos son de ánodo y cátodo común, siendo que se estaba usando el incorrecto.

Por otro lado, se observaron los detalles para conectar correctamente cada tipo de componente. Los LEDs se conectan con el ánodo al lado positivo y cátodo a la tierra. Para el display, los pines de alimentación controlan el encendido y apagado de cada segmento, por lo que el código debe controlar el orden de encendido de los segmentos. También es importante resaltar la importancia de las resistencias, porque sin ellas los componentes se quemarían. Finalmente, los push buttons sirven para mandar una señal e iniciar una respuesta del circuito.
