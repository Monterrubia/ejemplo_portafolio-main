La práctica 3 tiene como objetivo comenzar a usar el Attiny85. Este es un microcontrolador que permite subirle el código para luego conectarlo al circuito y no controlarlo mediante el arduino directamente.

Práctica 4: Attiny85 Outputs

La práctica 4 comprende el uso de nuevos componentes con el Attiny85. Dichos componentes son los siguientes outputs: LED, LED RGB, LED neopixel, LCD 12x16, buzzer, motor DC y servomotor. De la misma manera, a este microcontrolador se le continúa cargando el código para no necesitar de una conexión USB a la computadora al momento de ejecutar el código.
